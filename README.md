# 📝 Productivity Tracker

A simple full-stack productivity app built with **React** (frontend) and **Node.js + Express** (backend). Users can add tasks, set due dates, mark tasks as complete, and remove them.

---

## 🚀 Features

- Add tasks with a due date
- Mark tasks as complete/incomplete
- Delete tasks
- Real-time updates without page refresh
- Clean and minimal UI

---

## 🛠 Tech Stack

| Layer      | Tech              |
|------------|-------------------|
| Frontend   | React             |
| Backend    | Node.js + Express |
| Styling    | Inline CSS (Basic) |
| API Format | RESTful JSON      |
